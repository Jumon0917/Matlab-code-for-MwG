# Matlab-code-for-MwG
This is Matlab code for the study 'A hierarchical Bayesian approach for time-varying (possible multiple setups) finite element model updating'
The two programs beginning with 'main' correspond to the first and second examples in the paper, respectively. 'MwGsample' is the main sampling program, in which some parameters need to be adjusted according to the actual situation of the example.
